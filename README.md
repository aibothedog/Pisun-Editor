### How to create themes?
By using new feature "Custom themes" we can create own themes
##### Theme example:
```
<Name of theme>
<Main color>
<Secondary color> (for menu bar, etc)
<For text (Foreground)>
```

##### Black theme example:
```
Black
black
gray7
white
```

While creating a theme, use tkinter color pallete, or HTML Hex colors
